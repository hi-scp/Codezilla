Styling your button


Do you think the buttons we've used so far are ugly? Good news! You can make your buttons more stylish using CSS.

How do you enter CSS code? In the "head&head section of your file, use the opening and closing tags <style> and </style>. Inside these tags, you want to create a class for your button. A button class could look something like this:

.button {
  property1: value
  property2: value
  etc.
}

Replace "property#" with actual properties provided by CSS! You can look up really huge lists of CSS properties online, but here are some useful ones for buttons:

  • background-color: background color changes the color of your button! You can use a standard CSS color like black or red. A list of colors is available here at the W3schools site. This website is useful for other coding lessons as well! You can also type a color with the format "rgb(#, #, #)".
  • border: changes the border color of your button.
  • color: changes the text color of your button
  • text-align: changes the way your text is aligned within the button. The options for this include left, right, center, and justify.
  • font-size: self-explanatory! Format your font size value like this: "#px"
  • cursor: change the way your cursor appears when hovering over the button! You can see some cursor options, once again, at the W3schools site

Another cool feature is the ability to make your button change whenever your cursor hovers over it. This can be done by adding .button:hover {} in CSS which is just like the .button class mentioned above. Inside, simply list the attributes you wish to change while your cursor hovers over it.

Lastly, when you make your button in the body of your code, include 'class= "button"' in the opening tag (note that button is the name of your button class). For this exercise, you can try to replicate the button shown below, or just have fun creating your own button!
